# vocab-frontend

A Clojure library designed to learn Chinese Vocabulary.

## Usage

### Useful commands

Update dependencies ```lein deps```  \
Compile ClojureScript application ```lein cljsbuild auto dev``` \
Run python server ```python -m SimpleHTTPServer 8080``` \
